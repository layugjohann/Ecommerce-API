# E-Commerce API (Prototype)

## Description
This project is a prototype REST API for an e-commerce application.  
Users can browse products, add items to their cart, and place orders.

The API handles authentication, product management, cart operations, and order checkout.

The API is built using:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

This project is intended for learning backend development and REST API design.

---

## Features

- User registration and login
- JWT authentication
- Product creation and management
- Retrieve active products
- Add products to cart
- Update cart quantity
- Remove items from cart
- Checkout cart to create orders
- Prevent checkout with empty cart

---

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JSON Web Token (JWT)
- bcrypt
- CORS
- Dotenv
- Postman (API testing)
